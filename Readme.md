# 프로젝트 개요
## 주제
- 지역 별 중고 거래 카테고리 별 인기 물품 분석
- 중고 거래 사이트는 [당근마켓](https://www.daangn.com/kr/?in=%EA%B0%95%EB%82%A8%EA%B5%AC-381)을 이용함
- 모든 지역이 아닌 강남구로 특정하여 진행함

## 목표
-  당근마켓에서 지역별 인기 카테고리 매물 분석 시각화
